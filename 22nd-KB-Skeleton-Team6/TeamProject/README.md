# KB Bank Budget App UI

This is a code bundle for KB Bank Budget App UI. The original project is available at https://www.figma.com/design/9ypLkoXZoRKUbGzuSMx0le/KB-Bank-Budget-App-UI.

## Running the code

1. Run `npm i` to install the dependencies.
2. Set the API base URL in `.env`:

   ```env
   VITE_API_BASE_URL=http://localhost:3001
   ```

3. Run `npm run server` to start json-server on port `3001`.
4. Run `npm run dev` to start the development server.

> Team standard port: `3001`
